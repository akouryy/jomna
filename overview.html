Jomna overview

syntax

numbers

Jomnaでは、数値は64進数で表されます。

01…9 ab…z AB…Z ~_
01…9 1011…35 3637…61 6263
70 # => 448 1a # => 74 1A # => 100 7_5~3 # => 133980035 z.w # => 35.5 数値リテラルの最初の文字は数字または英小文字である必要があります。 abc # OK 0abc # OK ABC # wrong 0ABC # OK

strings

Jomnaには2種類の文字列リテラルがあります。ダブルクォート文字列とシングルクォートシンボルです。

ダブルクォート文字列は、一般的な文字列の形式です。ダブルクォート文字(")で始まり、"やバックスラッシュ(\)以外の任意の文字(またはエスケープシーケンス、後述)が0個以上並び、"で終わります。
また、\から始まるエスケープシーケンスによって、特殊な文字を表現できます。

\a \b \e \f \n \N \r \s \t \c? \m? \M? \??? \x?? \u???? \?
BEL BS ESC FF LF CRLF CR SP TAB Ctrl-? Meta-? Meta-Ctrl-? chr 0??? chr 0x???? ?
?:任意の文字 0-7 0-9a-fA-F 左記以外
"hello" "Pyon\tPyon\n" "\"\'\\\0\012\x12\u1234" "ab c de fg" # any char other than " or \ can be put without escaping

シングルクォートシンボルは、シングルクォート(')で始まり英数字もしくはエスケープシーケンスが1個以上並んだ形式です。

'world\n 'with\sspace 'Pyon-Pyon # wrong 'Pyon_Pyon # wrong
